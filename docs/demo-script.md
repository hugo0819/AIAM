# 演示脚本 · 5 分钟逐字稿

> 面向评委的现场演示口播稿。建议提前录屏，作为现场翻车兜底。
>
> 角色：**小李**（持有人 / 演示者），**Aria**（AI 行政助理 / 剧本化执行）。

---

## 0. 开场（30s）

> 「大家好，我这次想回答一个问题——**当 AI Agent 真正开始替我们干活，我们拿什么管住它？**
>
> OAuth 是为人设计的，RBAC 是为应用设计的，但 Agent 是**代人行动、会调 Agent、会访问敏感数据、行为还非确定**的新物种。
>
> 所以我们做了一个叫 **AI Agent Passport** 的东西——给每一个 AI，发一张数字通行证。」

**画面**：首页 Hero，四个锚点 C1~C4 卡片，slogan「可控 · 可审 · 可追责」。

---

## 1. S1 · 签发通行证（45s · 锚点 C1）

> 「这是我的 AI 助理 Aria。现在我给她签一张通行证。」
>
> 「这张通行证不是一个简单的 token，而是一份**结构化声明**——
> 她能用哪些工具、访问哪些数据、在什么时间窗、什么地理范围、单次金额上限多少，
> 全部都在这张 JWT 里，用 ES256 签名，任意网关都可以验签。」

**画面**：控制台 → 通行证 → 签发表单（能力勾选、约束配置），点击签发后**通行证卡片**淡入。

**关键台词**：
- 「注意这里有一条：**expense.submit 单笔金额 ≤ 500 元**。」
- 「这是整个系统的视觉符号——一张可见的通行证。」

---

## 2. S2 · 正常调用（30s · 基础能力）

> 「先走一个正常路径。让 Aria 帮我约一个明天下午的会。」

**画面**：演示舞台 → S2 → 播放 → 右侧审计时间线出现 10 步事件（RECEIVED → VERIFIED → POLICY_CHECKED → MASKED → EXECUTED → LOGGED）。

**关键台词**：
- 「每一次工具调用，网关都会生成一条完整的审计链——**你永远知道是谁在什么时间做了什么**。」

---

## 3. S3 · 数据脱敏（45s · 锚点 C2）

> 「接下来演示我们的第二个核心创新——**数据平面的管控**。」
>
> 「很多 AI 安全方案到'权限通过了就放行'就停了。但 Agent 拿到数据就会进 LLM 上下文，
> 上下文里夹带 PII 是非常危险的。所以网关还要拦数据。」

**画面**：S3 播放，展开一个事件详情，左右对比：
- 左：原始出参 `13812341234 / 330102199001010012`
- 右：脱敏后 `138****1234 / 3301**********0012`

**关键台词**：
- 「**LLM 永远拿到的是右边这个版本**，但审计日志留的是左边的版本供追溯。」

---

## 4. S4 · 越权拦截（20s · 基础能力）

> 「如果 Aria 想干超出能力的事呢？比如她突然想看同事薪资。」

**画面**：S4 播放，红色 DENY 动画，命中策略 ID 高亮。

**关键台词**：
- 「三道防线依次命中：能力匹配失败 → 策略引擎拒绝 → 审计留痕。毫秒级。」

---

## 5. S5 · JIT 二次确认（50s · 锚点 C3）

> 「第三个创新：**即时风险升级**。」
>
> 「传统思路是事先把所有权限给足，风险就大。我们的思路是——
> **不给多，到需要的时候当场让人来确认。**」

**画面**：S5 播放 → Aria 提交 600 元报销 → 模拟手机推送弹窗出现在另一个窗口 → 小李点「批准」→ 主流程恢复。

**关键台词**：
- 「刚才那张通行证写的是 ≤500，现在这笔是 600，网关拦下来，**给我手机推送了一条**。」
- 「我批准，调用继续；我拒绝，就结束；60 秒没响应，默认拒绝。」
- 「审计日志上这一步会被标记为**经人工核准**。」

---

## 6. S6 · 委托链（45s · 锚点 C4）

> 「最后一个创新点——**信任链路的可视化**。」
>
> 「Aria 要订机票，她不自己做，而是调用一个叫'差旅子 Agent'的助手。
> 这时候通行证会**派生**出一张**能力是父集子集**的子通行证。」

**画面**：S6 播放 → 信任图谱实时绘制三层节点（小李 → Aria → 差旅 Agent → flight.book），颜色渐变。

**关键台词**：
- 「你看图里的这条链路：每一步都有身份、每一次调用都可追。**责任归属一目了然。**」

---

## 7. S7 · 异常 + 吊销（30s · 生命周期）

> 「那如果 Aria 被劫持了呢？」

**画面**：S7 播放 → 告警闪烁 → 小李点「一键吊销」→ 下一次调用立即失败。

**关键台词**：
- 「风险中心检测到凌晨 3 点的异常拉取，我一键吊销——
- **吊销名单是内存级的，派生通行证级联失效**，下一次调用毫秒级被拒。」

---

## 8. 收尾（15s）

> 「所以这就是 AI Agent Passport：
>
> **签发 → 运行时鉴权 → 数据脱敏 → 即时审批 → 审计追溯 → 吊销下线**，
> 一条完整的 AI 治理闭环。
>
> 让每一个 AI Agent 的每一次行动，都可控、可审、可追责。
>
> 谢谢。」

**画面**：回到首页，stop。

---

## ✦ 现场 Tips

1. **不要接真 LLM**：所有 Agent 行为走剧本，保证不翻车。
2. **提前清空数据库**：演示前跑 `npm run reset-db`（Day 2 实装）。
3. **两个浏览器窗口**：主屏控制台 + 副屏「小李手机」审批页。
4. **录屏兜底**：提前录 720p/1080p 版本，现场翻车切播放。
5. **PPT 最后一页**：贴一张 Passport 的 JWT Payload JSON，这是灵魂道具。
